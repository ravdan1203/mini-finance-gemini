# Mini Finance Gemini Receipt App

This is a Next.js app for receipt image upload and automatic finance transaction creation using Gemini API.

## Files

- package.json
- app/layout.js
- app/page.js
- app/globals.css
- app/api/gemini-receipt/route.js
- .env.example

## Vercel Environment Variables

Set these on Vercel, not in GitHub:

```text
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-2.5-flash
```

## Important

Do not upload `.env.local` to GitHub.
